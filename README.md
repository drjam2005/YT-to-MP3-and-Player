# YT to MP3 and player

This app utilizes `yt_dlp` to extract, download, and conert youtube videos into mp3s and store them locally, and doubles as an MP3 player



## Usages

- **Downloader**: 
	mp3 downloader into disk

- **MP3 player**:
  well, mp3 player

## Dependencies
- yt_dlp
- ttkbootstrap
- tkinter
- mutagen
