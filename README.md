# YT to MP3 and player

This app utilizes the `yt_dlp` API to extract and download youtube videos into mp3s and store them locally



## Usages

- **Downloader**: 
	mp3 downloader into disk

- **MP3 player**:
  well, mp3 player

## Dependencies (for exe)
- nothing

## Dependencies (for code shit)
- yt_dlp
- ttkbootstrap
- tkinter
- mutagen
